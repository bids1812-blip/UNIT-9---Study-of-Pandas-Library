# STUDY OF PANDAS LIBRARY IN PYTHON

## AIM
To study and understand the Pandas library in Python and analyze its role in data manipulation, analysis, and structured data handling.

## THEORY

Pandas is an open-source Python library used for data analysis and data manipulation. It provides powerful and flexible data structures that make working with structured data easy and efficient.

Pandas is built on top of the NumPy library and is widely used in data science, machine learning, statistics, and research fields. It allows users to handle large datasets, clean data, filter information, and perform analytical operations with ease.

The two main data structures in Pandas are:

1. Series

2. DataFrame

These structures help in organizing and analyzing data in tabular form.

### Series

A Series is a one-dimensional labeled array capable of holding data of any type such as integers, strings, or floating-point numbers.

Each element in a Series has an associated index. It is similar to a single column in a table.

Series are commonly used for:

1. Storing one-dimensional data

2. Performing statistical calculations

3. Handling indexed data

### DataFrame 

A DataFrame is a two-dimensional labeled data structure with rows and columns. It is similar to a spreadsheet or a database table.

Each column in a DataFrame can contain different data types. DataFrames are highly flexible and support various operations such as filtering, sorting, grouping, and aggregation.

DataFrames are commonly used for:

1. Storing tabular data

2. Data cleaning and preprocessing

3. Data analysis and summarization

4. Handling CSV and Excel files


## FEATURES OF PANDAS

1. Efficient handling of large datasets.

2. Powerful data filtering and selection capabilities.

3. Support for reading and writing multiple file formats.

4. Built-in statistical and analytical functions.

5. Easy handling of missing data.

6. Integration with NumPy and other data science libraries.

These features make Pandas one of the most important libraries in data analysis.

## APPLICATIONS OF PANDAS

Pandas is widely used in:

1. Data cleaning and preprocessing

2. Business analytics

3. Financial data analysis

4. Machine learning projects

5. Research and statistical analysis

6. Data visualization preparation

It is an essential tool for working with real-world datasets.

## ADVANTAGES OF PANDAS

1. Simplifies complex data manipulation tasks.

2. Provides fast and efficient operations on structured data.

3. Improves productivity in data analysis projects.

## COMMON ERRORS

1. Incorrect column or index referencing.

2. Mismatch in data dimensions.

3. Improper handling of missing values.

4. Confusion between Series and DataFrame structures.

5. Logical mistakes while filtering data.

6. Understanding the structure of data properly helps in avoiding these errors.
